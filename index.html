Tentang Saya

Saya adalah siswa SMK jurusan Rekayasa Perangkat Lunak yang sedang mempelajari pengembangan website. Saya terbiasa menggunakan HTML dan CSS untuk membuat tampilan website, serta PHP untuk mengelola data dan logika aplikasi sederhana.

Dalam proses belajar, saya mengerjakan berbagai tugas dan proyek sekolah seperti website CRUD, sistem login, dan pengelolaan data menggunakan MySQL dan PostgreSQL. Saya juga mulai menggunakan framework Laravel dan CodeIgniter 4 untuk memahami struktur aplikasi web yang lebih terorganisir.

Selain web, saya juga mengenal dasar pengembangan aplikasi menggunakan Android Studio. Saya senang belajar hal baru di bidang teknologi dan terus berusaha meningkatkan kemampuan saya melalui latihan dan eksplorasi mandiri.

Cara Saya Bekerja

Saya bekerja dengan prinsip sederhana: rapi, fungsional, dan dapat dipertanggungjawabkan. Setiap baris kode ditulis agar mudah dibaca dan dikembangkan.

Saya terbiasa bekerja mandiri maupun dalam tim, menggunakan tools modern untuk memastikan efisiensi dan konsistensi.

Bagi saya, website yang baik adalah website yang terasa mudah digunakan tanpa perlu penjelasan panjang.
