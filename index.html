Tentang Saya

Halo! Saya Aleza Andhika, seorang siswa SMK jurusan Rekayasa Perangkat Lunak yang memiliki passion besar di dunia pengembangan web.

Saya mulai belajar coding sejak tahun 2023 dan sejak saat itu saya terus mengembangkan skill saya di berbagai teknologi web. Saat ini saya fokus mempelajari PHP dengan framework Laravel dan CodeIgniter 4, serta memperdalam pemahaman tentang database MySQL dan PostgreSQL.

Bagi saya, menjadi developer bukan hanya tentang menulis kode, tapi juga tentang memecahkan masalah dan menciptakan solusi yang bermanfaat. Saya selalu terbuka untuk belajar hal-hal baru dan berkolaborasi dalam proyek-proyek menarik.
